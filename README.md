# PG Life Clone

This is a full-stack web application inspired by the PG Life project.

## Tech Stack
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap
- React (for property listing module)

## Project Overview
PG Life Clone is a platform where users can browse PG accommodations based on city, view property details, and mark properties as interested.

## Project Structure
The complete source code is available inside the `PGLifezip.zip` file.

Due to the presence of large image assets, the project has been uploaded as a ZIP file.

## How to Run the Project Locally
1. Download and extract `PGLifezip.zip`
2. Move the extracted folder to `xampp/htdocs`
3. Start Apache and MySQL from XAMPP
4. Import the database into phpMyAdmin
5. Update database credentials in `includes/config.php`
6. Open browser and go to:
   http://localhost/PGLife

## Notes
- Database credentials are not included for security reasons.
- This project was built for learning and practice purposes.
